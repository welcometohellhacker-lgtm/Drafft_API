# Drafft_API

Drafft is a B2B AI-powered video optimization platform for mortgage, insurance, and financial service companies. Users upload long-form videos, from about 1 minute up to 1 hour, and the system automatically transforms them into high-engagement short clips with captions, B-roll, transitions, CTA overlays, optional narration, and export-ready rendered videos.

The platform must be designed as a modular pipeline, with all earlier versions included as milestones, but the final implementation target is Version 5.
